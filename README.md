# IoTを用いた環境測定  
## TL;DR
- 温湿度などをIoT機器で測定
- 測定データをWebで閲覧できる

## ハードウェア
- [wioterminal](https://wiki.seeedstudio.com/jp/Wio-Terminal-Getting-Started/)
- [Raspberry Pi4 Model B 4G](https://jp.rs-online.com/web/generalDisplay.html?id=raspberrypi)*サーバーとして利用

## ミドルウェア
- Nginx
- MySQL

## 言語
- PHP(サーバーサイド)
- arduino(IoT機器)
